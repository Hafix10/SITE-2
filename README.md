# Mercedes-Benz  Page

A responsive landing page using HTML and CSS. The project focuses on creating a clean, while applying basic responsive design techniques to ensure a better viewing experience across different screen sizes.

## Features

* Responsive layout for desktop and mobile devices
* Clean and organized HTML & CSS code
* Simple and user-friendly interface

## Technologies Used

* HTML5
* CSS3

## Purpose

This project was created for practicing front-end web development and improving responsive web design skills.
